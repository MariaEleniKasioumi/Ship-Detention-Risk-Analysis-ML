# Ship-Detention-Risk-Analysis-ML
Paris MoU Data

- Reproduced and extended a published risk analysis model using maritime inspection data (Paris MoU)  
- Processed and transformed inspection data (cleaning, categorization, feature engineering)  
- Applied association rule mining (support, confidence, lift) to identify defect patterns  
- Built and evaluated a Random Forest model for ship detention risk classification  
- Applied PCA and hyperparameter tuning (Grid Search) to improve model performance  
- Interpreted model predictions using SHAP to identify key risk factors  
- Compared results with the original study and proposed improvements
